# Tim-Gi-Oh Premium Linktree Alternative

Dieses Repository enthält den optimierten, barrierefreien Quellcode für die Landingpage von **Tim-Gi-Oh**. Das Projekt wurde als performante und werbefreie Alternative zu Drittanbieter-Plattformen (wie NexLink) entwickelt.

## Features & Optimierungen

* **Keine Ladezeiten:** Basiert auf statischem HTML5 und Tailwind CSS. Lädt ohne Verzögerung (keine Preloader).
* **Auto-Copy für Rabattcodes:** Beim Klick auf die Partner-Buttons (ICE Sleeves, Steelduck, Jungcards) wird der jeweilige Code automatisch in die Zwischenablage kopiert.
* **Barrierefreiheit:** Semantische HTML-Struktur, optimiert für Screenreader.
* **Datenschutzkonformes Impressum:** Rechtliche Angaben öffnen sich direkt als schnelles Overlay auf der Seite, ohne die Nutzer umzuleiten.
* **100% Werbefrei:** Kein fremdes Branding oder Tracker von Drittanbietern.

## Deployment-Optionen für das Tech-Team

Das Projekt kann auf zwei Wegen live geschaltet werden:

### Option A: Forken & Eigenes Hosting (Empfohlen für volle Kontrolle)
1. Dieses Repository forken.
2. Die `index.html` auf euren eigenen Webspace hochladen oder via GitHub Pages / Vercel direkt mit eurer Wunsch-Domain (z. B. `tim-gi-oh.de/links`) verknüpfen.
3. Die Domain im `data-domain` Attribut des Plausible-Analytics-Skripts im `<head>` anpassen.

### Option B: Hosting über meine Infrastruktur
Falls gewünscht, kann das Repository direkt über meine GitHub-Organisation laufen. Ich muss lediglich die Custom-URL auf eure Wunsch-Domain umstellen.
